Main Features of Transible

Free and Open-Source

Transible is free and open-source software. It is distributed under the terms of the Apache License, Version 2.0.

Multicloud Support

Transible supports all major cloud providers: Azure, AWS, Openstack and more to come.

Simpplicity

Transible is simple to use. It is based on Ansible, so you can use all its features.

Synergy

Transible adheres to the best practice of infrastructure as code and cloud infrastructure in mind.
